IfcStructuralLoadSingleForce
============================
Instances of the entity _IfcStructuralLoadSingleForce_ shall be used to define
the forces and moments of an action operating on a single point.  
  
> HISTORY  New entity in IFC2x2.  
[ _bSI
Documentation_](https://standards.buildingsmart.org/IFC/DEV/IFC4_2/FINAL/HTML/schema/ifcstructuralloadresource/lexical/ifcstructuralloadsingleforce.htm)


Attribute definitions
---------------------
| Attribute   | Description                 |
|-------------|-----------------------------|
| ForceX      | Force value in x-direction. |
| ForceY      | Force value in y-direction. |
| ForceZ      | Force value in z-direction. |
| MomentX     | Moment about the x-axis.    |
| MomentY     | Moment about the y-axis.    |
| MomentZ     | Moment about the z-axis.    |
