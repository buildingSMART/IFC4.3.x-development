IfcVolumeMeasure
================
An _IfcVolumeMeasure_ is the value of the solid content of a body.  
  
Usually measured in cubic metre (m3).  
  
Type: REAL  
  
> NOTE  Type adapted from **volume_measure** defined in ISO 10303-41.  
  
> HISTORY  New type in IFC1.5.1.
